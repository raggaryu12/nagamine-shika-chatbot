このチャットボットについて

Anthropic Claude API を使用した、ながみね歯科クリニック専用AIアシスタントです。 診療内容・アクセス・予約方法・治療費のご案内など、患者様からのよくあるご質問に 丁寧な言葉でお答えします。

ストリーミング回答 会話履歴保持 スマートフォン対応 埋め込み可能 24時間対応

セットアップ手順

1
依存パッケージのインストール 
npm install
2
.env ファイルの作成 
# .env ファイルを作成して API キーを設定
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxx
PORT=3000
3
サーバーの起動 
node server.js
4
ブラウザで確認

http://localhost:3000 にアクセスし、 右下のチャットアイコンをクリックしてください。

既存サイトへの埋め込み方法

以下のコードを既存のホームページの </body> タグ直前に追加するだけで埋め込めます。

<!-- APIのURLをサーバーのURLに変更してください -->
<script>
  window.NAGAMINE_CHATBOT_API_URL = 'https://your-server.com/api/chat';
</script>
<link rel="stylesheet" href="https://your-server.com/chatbot.css">
<script src="https://your-server.com/chatbot.js"></script>
※ サーバーを HTTPS で公開する場合は、Render・Railway・Heroku などのホスティングサービスをご利用ください。
